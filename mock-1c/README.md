# mock-1c

Node.js HTTP-сервис, имитирующий внешний источник учебных данных (1С). Не имеет внешних зависимостей — только стандартная библиотека Node.

## Запуск

```bash
node server.js
```

Сервис поднимается на `http://127.0.0.1:8091`.

## Эндпоинты

```
GET /onec/students/{id}/profile
GET /onec/students/{id}/grades
GET /onec/students/{id}/schedule
GET /onec/students/{id}/debts
```

Демо-студенты: `2021-301-047`, `2021-301-048`, `2022-402-015`, `2020-104-033`.

## Веб-интерфейс

```
http://localhost:8091/admin
```

Позволяет добавлять, изменять и удалять оценки студентов без перезапуска сервиса. Изменения попадают в личный кабинет после очередной синхронизации `student-api` (по умолчанию каждые 10 минут) или после перезапуска бэкенда.

## Роль в проекте

`mock-1c` нужен только для локальной разработки. `student-api` забирает данные из него при старте и по расписанию через `OneCSyncService`. Фронтенд с `mock-1c` напрямую не взаимодействует.
